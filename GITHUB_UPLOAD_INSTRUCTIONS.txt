GitHub-only upload package

Upload these files together to the SAME folder in your GitHub repository:
- index.html
- uae_local_boundary_pack.geojson
- UAE_Emirates_Boundaries.geojson
- uae_digital_map_style.json
- uae_interactive_points.geojson

Then enable GitHub Pages on:
- Branch: main
- Folder: /(root)

Open your site after Pages finishes building.

This package is static-only:
- no PHP
- no XAMPP
- no proxy file required
